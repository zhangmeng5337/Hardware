# OTA

#### 介绍
本项目是STM32+4G模组的OTA升级方案。STM32嵌入式程序包含BOOT(搬运APP2的程序到APP1的位置)，APP1（运行的程序），APP2（将要升级的程序）。APP的程序使用AT指令和4G交互。下载升级的BIN文件使用http服务。
